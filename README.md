\# File Summarizer



A simple Python CLI tool that reads any `.txt` file and prints a word count, line count, and first 3 lines preview.
Right now it can only read .txt files. im working to make it read other filetypes too :)



Built as a mini scripting exercise to practice file I/O and CLI argument handling.



\## Usage

```bash

python3 summarize.py yourfile.txt

```



\## What I learned

\- Reading files with open() and readlines()

\- Handling CLI args with sys.argv

\- Basic input validation
